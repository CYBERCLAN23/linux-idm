# Contributing to Linux IDM

Thank you for your interest in contributing to Linux IDM! We welcome contributions from everyone.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and include steps to reproduce it.
- Mention your Linux distribution and browser version.

### Suggesting Enhancements
- Open a new issue and describe the feature you'd like to see.
- Explain why this feature would be useful.

### Pull Requests
1. Fork the repository.
2. Create a new branch: `git checkout -b feature/my-new-feature`.
3. Commit your changes: `git commit -m 'Add some feature'`.
4. Push to the branch: `git push origin feature/my-new-feature`.
5. Open a Pull Request.

## Code of Conduct
Please be respectful and professional in all interactions.
